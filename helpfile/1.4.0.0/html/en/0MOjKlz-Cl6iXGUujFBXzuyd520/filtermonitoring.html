FilterMonitoringB (FB)¶

FUNCTION_BLOCK FilterMonitoringB

Functional Description¶

Detection of a filter contamination and generation of a fault message

Typical Application: Detection of air filter contamination with a differential pressure switch

Functional Description¶

General¶

This function block is used to detect filter contamination by analyzing the input for the
differential pressure switch xIn.

Output - fault message xAlarm¶

The output - fault message xAlarm is activated (xAlarm = TRUE) if condition 1 below is met.

Condition 1: The input for the differential pressure switch xIn is permanently in the TRUE (=filter contamination ) state at least during the waiting time udiControlTime.
At the same time the detection is released generaly (xContol = TRUE).

The output - fault message xAlarm is deactivated (xAlarm = FALSE) in following cases:

Case 1: The Condition 1 is not met and the reset of the fault message xQuit is active (= TRUE).

Case 2: The Condition 1 is not met and the reset of the fault message xQuit is inactive (= FALSE).

Time duration until the fault message is triggered * tTimeToAlarm*¶

The output time duration is triggred until the fault message tTimeToAlarm is released. Depending on the operation situation different values are shown.

Situation 1: Output of the remaining time duration until the fault message is released (Condition: xIn = TRUE, xControl = TRUE)

Situation 2: If the situation 1 is not present, then the value 0.0 is permanently displayed.
