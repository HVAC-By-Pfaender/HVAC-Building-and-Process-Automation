Min8Fun (FUN)¶

FUNCTION Min8Fun : REAL

Kurzbeschreibung¶

Minimalwertbildung aus 8 nummerischen Eingängen

Typische Anwendung: Berechnung der tiefsten Temperatur

Funktionsbeschreibung¶

Diese Funktion liefert den Minimalwert aus den 8 nummerischen Eingängen rValue1 bis rValue8 im Format REAL zurück.
