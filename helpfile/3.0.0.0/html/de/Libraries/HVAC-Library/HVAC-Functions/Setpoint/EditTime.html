EditTime (FB)¶

FUNCTION_BLOCK EditTime

Kurzbeschreibung¶

Eingabe einer Zeitdauer im Format TIME ( t#12h34m15s )

Typische Anwendung: Festlegung einer Zeitverzögerung

Darstellung¶

Schnittstellen¶

Ausgänge¶

Name Datentyp Wertebereich Initialwert Funktion
tOut TIME     Ausgabewert im Format TIME

Sollwerte / Parameter¶

Name Datentyp Wertebereich Initialwert Funktion
tSetPoint TIME   T#0s Eingabewert im Format TIME

Funktionsbeschreibung¶

Allgemeines¶

Dieser Funktionsbaustein dient zur Übergabe eines Eingabewerts im Format TIME tSetPoint und die direkte Übertragung zum Ausgabewert im Format TIME tOut.

Verhalten bei Spannungsausfall

Der Eingabewert im Format TIME tSetPoint wird spannungsausfallsicher gespeichert und automatisch wiederhergestellt.
