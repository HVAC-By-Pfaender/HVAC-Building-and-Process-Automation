SystemInfo.mGetNetworkInfo (METH)
---------------------------------

METHOD mGetNetworkInfo




Beim Aufruf der Mehode, werden die Ausgänge für die Netzwerkinformationen aktualisiert.
Die Methode wird außerdem auch von der SystemInfo Instanz aufgerufen, sobald der Eingang "xGetNetworkInfo" getriggert wird.
