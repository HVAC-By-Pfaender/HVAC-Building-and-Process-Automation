AlarmCounter (FB)¶

FUNCTION_BLOCK AlarmCounter

Short Description¶

Read active alarms from a given alarm group or alarm class.

Typical application: Displays the number of active alarms.

Functional Description¶

General¶

This function block is used to read out the number of active alarms in different states.
The alarm counter can be filtered by an alarm group or alarm class, as well as by a combination of these two options.
If no filter of the available filter options is specified, all alarms of the respective option are considered with the counter.


Example configuration 1:
The two inputs sFilterByAlarmGroup and sFilterByAlarmClass are not specified.

Effect of the example configuration 1:
All alarms are considered regardless of the alarm class and alarm group.

Example configuration 2:
The input sFilterAlarmClass is assigned the STRING “Alarm” and the input sFilterByAlarmGroup remains unassigned.

Effect of the example configuration 2:
All alarms with alarm class “Alarm” are displayed by all alarm groups.
