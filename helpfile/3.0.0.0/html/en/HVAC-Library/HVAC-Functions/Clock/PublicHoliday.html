Functional Description¶

General¶

This function block determines whether a public holiday is active and displays the name of the public holiday.
The function block must be activated via input xEn (TRUE = function active).
Holidays can be defined using the stPublicHoliday structure. Here you specify the name, the date of the public holiday and the release status.
Holidays can be defined as a fixed date, with an interval to Easter or weekdays before a fixed date.
The setpoint / parameter eCountry allows you to define the country whose holidays are to be used. Currently only the holidays for Germany are implemented.
