DamperAnalog (FB)¶

FUNCTION_BLOCK DamperAnalog

Short Description¶

Control of a continuous damper actuator with continuous position feedback

Typical application: Continuous recirculation damper of a HVAC system

Functional Description¶

Actuator signal Damper actuator before manual override rAA¶

The control signal of the damper actuator before manual override rAA is accepted unchanged from the input set point - position rY.

Control signal Damper actuator after manual override rHA¶

The control signal of the damper actuator after manual override rHA corresponds to the control signal of the damper actuator before manual override rAA, additionally extended by a manual override module and a forced override.

xAlarm collective alarm¶

The collective alarm xAlarm is activated ( = TRUE ) if the amount of the difference between the control signal damper actuator after manual override rHA and the actual value of position rU permanently exceeds the maximum deviation between setpoint and actual value of position rControlDiff for a minimum period of udiControlTime.
At the same time the release of the position monitoring xControl must be active ( = TRUE ) and the operating mode of the manual override hardware must be in the HVACTYPES.eManBin.Auto position.

In all other cases, the xAlarm collective fault is not activated (= FALSE).

Collective message automatic mode xAuto¶

The collective message Automatic mode xAuto is activated ( = TRUE ) if the input eAO contains the value HVACTYPES.eManNum.Auto and the setpoint / parameter eManModeN contains the value eMANNUM.Auto. In all other cases, the collective message Automatic mode xAuto is not activated ( = FALSE ).

Forced override xForce¶

The forced override affects the output rHA directly. The parameter xForceOption defines the state of the output rHA when TRUE at the input xForce.

With xForceOption = FALSE
If the input xForce is TRUE, the output rHA is set to 0.

With xForceOption = TRUE
If the input xForce is TRUE, the output rHA is set to 100.
