Shade (FB)¶

FUNCTION_BLOCK Shade

Short Description¶

Control of a Venetian shade drive with open/close commands and central control
Additional functions: Slat adjustment/central control/comfort functions
Typical application: Control of a standard Venetian shade drive

Functional Description¶

General¶

This function block controls a shutter drive with open (xUp) and close (xDown) commands.
The open and close commands can be overridden by the parameters eManModeUp or eManModeDown independently of the automatic functions.
It is operated by the button inputs xButtonUp and xButtonDown.
Central operation is possible via the inputs xCentralUp and xCentralDown.
The duration of the operation of the inputs xButtonUp and xButtonDown determines the executed function ( duration < tButton: complete opening/closing, duration >= tButton: Blade adjustment ).
In addition, a forced control of the outputs xUP and xDown takes place, i.e. if the output xUp is active ( = TRUE ), the output xDown is deactivated ( = FALSE ) at the same time. This also applies to the reverse case.
If one of the two buttons ( xButtonUp = TRUE and / or xButtonDown = TRUE ) is pressed during a complete opening / closing, the opening / closing is interrupted immediately.

Central control¶

Each rising edge at the request - shade opening input via central control xCentralUp triggers complete opening of the shade ( xUp = TRUE and xDown = FALSE during the period tFullMovement ).
Each rising edge at the request - shades closing input via central control xCentralDown triggers complete closing of the shades ( xDown = TRUE and xUp = FALSE during the period tFullMovement ).
If the inputs for the central control are permanently active (= TRUE), no operation is possible with the xButtonUp and xButtonDown buttons.

Waiting period of switching between opening and closing tWaitReverse¶

Switching from opening to closing the shade and vice versa can be delayed by this setpoint/parameter.
The value 0.0s allows immediate switching.

Slat adjustment¶

The slat adjustment is triggered by continuous actuation of the xUp or xDown operating buttons for a period longer than the period required to distinguish between slat adjustment / complete opening/closing tButton.
While the respective button is pressed, a symmetrical output signal ( pause duration = pulse duration = tPulse ) is generated at the assigned output.

Full opening/closing¶

The complete opening/closing is triggered by pressing the respectively assigned xUp or xDown keys for a period less than the time required to distinguish between slat adjustment / complete opening/closing tButton.
At the corresponding output, a signal (state = TRUE) with a duration of tFullMovement is then generated without further influencing.
At the other output, no signal (state = FALSE) is generated during this period.

Collective message automatic mode xAuto¶

The collective message automatic mode xAuto is activated ( = TRUE ) if the following conditions are fulfilled at the same time:

- Eingang eAOOUp = HVACTYPES.eManBin.Auto
- Eingang eAOODown = HVACTYPES.eManBin.Auto
- Sollwert / Parameter eManModeUp = eMANBIN.Auto
- Sollwert / Parameter eManModeDown = eMANBIN.Auto
