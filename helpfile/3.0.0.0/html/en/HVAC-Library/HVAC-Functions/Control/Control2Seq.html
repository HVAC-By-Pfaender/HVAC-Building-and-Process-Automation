Control2Seq (FB)¶

FUNCTION_BLOCK Control2Seq

Short Description¶

Sequence control with switch-on/switch-off threshold for an analog input signal
The direction of action (inverted/direct) is also adjustable.

Functional Description¶

Digital output xY¶

The digital output xY is influenced by the direction of action eAction, the analog input value rIn and the two switching thresholds rSwitchOffPoint and rSwitchOnPoint.
If the direction of operation eControlMode.Direct is selected, the switch-on threshold rSwitchOnPoint must be greater than the switch-off threshold rSwitchOffPoint.
If the direction of operation eControlMode.Inverted is selected, the switch-on threshold rSwitchOnPoint must be smaller than the switch-off threshold rSwitchOffPoint.
eAction rIn xY

Notes
eControlMode.Direct rIn < rSwitchOnPoint und rIn* > rSwitchOffPoint

unchanged

Direction of action = Direct: acutal value doesn’t exceed thresholds

unchanged

eControlMode.Direct rIn <= rSwitchOffPoint FALSE

Direction of action = Direct: acutal value exceed switch-off thresholds
eControlMode.Inverted rIn <= rSwitchOnPoint TRUE

Direction of action = Inverted: acutal value exceed switch-off thresholds
eControlMode.Inverted rIn >= rSwitchOffPoint FALSE

Direction of action = Inverted: actual value exceed switch-on thresholds
