Functional Description¶

General¶

This function block is used to display status information about a macro. To do this, the entire instance path is splitted and displayed according to the desired function.

Input sVersion¶

The input is currently not used.

Input and output xAlarm¶

The input and output are currently not used.

Input and output xAuto¶

The input and output are currently not used.

Output sTitle¶

The output sTitle is identical to the output sInstanceName and contains the current name of the macro.

Output sLine1¶

The output sLine1 displays the content of the setpoint / parameter sEditLine1.

Output sLine2¶

The output sLine2 displays the content of the setpoint / parameter sEditLine2.

Output sInstanceName¶

The output sInstanceName is identical to the output sTitle and contains the current name of the macro.

Output sInstancePath¶

The output sInstancePath contains the current instance path of the macro ( Device.Application.POU ).
