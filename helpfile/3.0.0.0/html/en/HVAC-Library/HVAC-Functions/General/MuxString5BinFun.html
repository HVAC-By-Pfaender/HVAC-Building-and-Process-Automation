MuxString5BinFun (FUN)¶

FUNCTION MuxString5BinFun : STRING

Short Description¶

Multiplexer for strings with binary selection ( 5 binary inputs )

Typical application: Selection of message texts

Functional Description¶

This function returns the string selected by the binary inputs xEn1 ( FALSE = not active, TRUE = active ) to xEn5 ( FALSE = not active, TRUE = active ) in STRING format ( sValue1 to sValue5 ). The input xEn1 always has the highest priority, the input xEn2 the second highest, and so on. If no input is active, an empty string ( ‘’ ) is returned.
