MuxBin5Fun (FUN)¶

FUNCTION MuxBin5Fun : REAL

Short Description¶

Multiplexer for numeric input variables with binary selection ( 5 binary inputs )

Typical application: switching of temperature sensors

Functional Description¶

This function returns the numeric input variable selected by the binary inputs xEn1 ( FALSE = not active, TRUE = active ) to xEn5 ( FALSE = not active, TRUE = active ) in REAL format ( rValue1 to rValue5 ). The input xEn1 always has the highest priority. The input xEn2 the second highest, etc... If no input is active, the value 0.0 is returned.
