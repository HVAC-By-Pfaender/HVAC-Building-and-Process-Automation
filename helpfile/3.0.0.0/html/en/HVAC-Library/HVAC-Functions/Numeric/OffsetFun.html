OffsetFun (FUN)¶

FUNCTION OffsetFun : REAL

Short Description¶

Addition of a numeric offset to a numeric input quantity

Typical application: Adaptation of a setpoint temperature

Functional Description¶

This function adds the numeric offset rOffset to the numeric input variable rValue and returns the result rounded to a maximum of 2 decimal places.
