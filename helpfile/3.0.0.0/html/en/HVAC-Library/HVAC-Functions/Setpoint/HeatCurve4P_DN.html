HeatCurve4P_DN (FB)¶

FUNCTION_BLOCK HeatCurve4P_DN

Short Description¶

Heating curve with four outdoor temperature points and two characteristic curves (night, day).

Functional Description¶


The flow set point temperature rSupplyTemp is calculated as a function of the current outdoor temperature rOutTemp and the selected characteristic curve ( night characteristic curve: xDay = FALSE // day characteristic curve: xDay = TRUE ).
The flow set point temperature rSupplyTemp is limited by adjustable absolute minimum ( rSupplyTempMin ) and maximum values ( rSupplyTempMax ).

The characteristic curves are defined and adjustable by four outside temperature points each (separately for the night and day characteristic curve).
Each point is defined by an outside temperature and an assigned flow set point temperature.
The four outdoor temperature points of a characteristic curve are connected by straight lines.

Limitation

rTo4 > rTo3 > rTo2 > rTo1
The eManModeN parameter allows the flow set point temperature rSupplyTemp to be switched from automatic to manual mode and vice versa.
In manual mode, the flow set point temperature rSupplyTemp takes over the value of the parameter rManValue.
