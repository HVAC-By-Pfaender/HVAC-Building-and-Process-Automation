SystemInfo (FB)¶

FUNCTION_BLOCK SystemInfo

Short Description¶

Acquisition of system information / input of individual project information.


Portrayal¶

Functional Description¶


This function block enables you to read the system information.
The static values that are written to the controller when an application is downloaded are displayed here.
Values are also displayed that can change at runtime.
These values can be queried within a definable interval and can also be read out if required.



Input xGetNetworkInfo¶


With each rising edge ( FALSE > TRUE ) the current network information are returned.
The values are returned via the outputs sEthernetAdapterIPAddress, sEthernetAdapterSubnetMask, sEthernetAdapterGateway and sEthernetAdapterMACAddress.

Input xGetRTUpTime¶


The current running time of the system is read with every rising edge.
The value is returned via the output tRuntimeUptime.
