PulseR (FB)¶

Functional Description¶

General¶

At the impulse output xQ, after each rising edge ( FALSE > TRUE ) at the binary input xIN and / or at the setpoint / parameter xTest a impulse with a duration tPT is generated at the impulse output xQ ( prerequisites: xReset = FALSE and no falling edge at input xIN ).

Each rising edge ( FALSE > TRUE ) at the reset input xReset immediately resets the impulse output xQ ( = FALSE ).
Each falling edge ( TRUE > FALSE ) at the binary input xIN immediately resets the impulse output xQ ( = FALSE ).

The edge detection ( FALSE > TRUE ) at the inputs xIN or xTest has a higher priority than the edge detection at the inputs xReset ( FALSE > TRUE ) or xIN ( TRUE > FALSE ).
As a result, a impulse with a time duration tPT is generated at output xQ despite simultaneous edge detection.

The setpoint / parameter xTest is deactivated cyclically ( = FALSE ).

Time output tET¶

If the impulse output xQ ( = TRUE ) is active, the current duration of the impulse at xQ ( 0.0 - tPT ) is displayed.
With inactive impulse output xQ ( = FALSE ) the value T#0s is always displayed.
